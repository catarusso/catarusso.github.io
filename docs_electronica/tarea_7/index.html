Tarea 7

Blink

Para este ejercicio se hizo parpadear un led con arduino. El código le indica al arduino que cada 1 segundo se prenda y se apague el led que está conectado al pin indicado (en este caso al pin 13).

Potenciómetro

Este ejercicio es muy similar al anterior, con la diferencia de que en este circuito se agregó un potenciómetro para regular la potencia de la luz y la velocidad de los parpadeos. Cuando el potenciómetro está conectado con 5 voltios a través de sus dos pines exteriores, el pin del medio entrega un voltaje entre 0 y 5, dependiendo de la posición de la perilla en el potenciómetro.

Led RGB

Un LED RGB o rojo-verde-azul tienen tres diodos emisores de color diferentes que se pueden combinar para crear todo tipo de colores. En este ejercicio se utiliza un LED RGB para ver como este alterna entre sus tres colores principales cada 1 segundo mientras emite luz.

Multiples Leds

En este ejercicio se debieron prender 5 leds seguidos. La idea es que estos leds se fueran prendiendo uno después del otro cada un segundo. Es parecido al ejercicio Blink pero esta vez con más de un led.

LDR

En este ejercicio, se utilizó una fotorresistencia, que cambia la resistencia en función de la cantidad de luz que el sensor recibe. Se debe utilizar un divisor de voltaje para que se pueda usar la fotoresistencia ya que el arduino no interpreta la resistencia como tal. Lo que hace el divisor es emitir un alto voltaje cuando recibe mucha luz y un voltaje bajo cuando no. Un alto voltaje hace que el led emita más luz y un bajo voltaje hace que emita menos luz.
